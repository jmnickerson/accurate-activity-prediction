As our homes become smarter, accurately predicting what the resident is doing can make our lives easier, cheaper and healthier.

What

Using data from the CASAS Project and machine learning techniques, this project accurately predicts what activity a person is engaged in based on sensor data. The machine learning was done using PyBrain with a feed-forward neural network.

Why

There are many exciting opportunities if a resident's activity can be accurately predicted. One such application is predicting power usage. This could allow for cost savings and power consumption reduction. Video

Quick Facts

  • ~12000 data points
  • 8 Hidden Layer Neural Network

How

The data from the CASAS project looks like this: 

2009-11-01 18:16:30.019905	M003	ON
2009-11-01 18:16:30.030466	M015	ON
2009-11-01 18:16:31.053482	M014	ON	Eating begin
2009-11-01 18:16:34.087702	M015	OFF
2009-11-01 18:16:36.021796	M003	OFF
2009-11-01 18:16:36.075721	M014	OFF
2009-11-01 18:16:45.071472	M003	ON
2009-11-01 18:16:46.025461	M014	ON

First the data is parsed to extract of which sensors were triggered during a specific activity. This data is written to a file as two boolean arrays, one for the sensors and one for the activities.

The array of triggered sensors is the input to the neural network and the array of activities is the output. The network was then trained over 10000 iterations with %80 of the data.
